# 氢能组态上位机系统 - 界面优化设计计划

## 项目概述
本计划基于 .NET 8 WinForms 项目，使用 SunnyUI 框架，参考 React/Tailwind 深色主题设计风格，对氢能组态上位机系统进行界面优化设计。

## 参考设计风格分析
从参考项目中提取的关键设计元素：
- **主题配色**：深色主题（slate-900/slate-950背景），cyan-400 主色调，红色/绿色/黄色作为状态指示色
- **布局结构**：顶栏 + 侧边栏导航 + 主内容区 + 底栏
- **视觉风格**：卡片式设计、圆角边框、清晰的信息层次

## 优化原则
1. 保持功能完整性和操作便捷性
2. 适当缩小界面元素尺寸，保持比例协调
3. 采用 SunnyUI 组件，代码写在 .Designer.cs 文件中
4. 维持与参考设计的风格一致性

## 配色方案（基于 SunnyUI 深色主题）
| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 主背景 | #0f172a (slate-950) | 主窗体背景 |
| 卡片背景 | #1e293b (slate-900) | 卡片和面板背景 |
| 主色调 | #22d3ee (cyan-400) | 标题、图标、激活状态 |
| 成功色 | #22c55e (green-500) | 运行状态、正常 |
| 警告色 | #eab308 (yellow-500) | 异常、警告 |
| 危险色 | #ef4444 (red-500) | 报警、停止 |
| 文字主色 | #f1f5f9 (slate-100) | 主要文字 |
| 文字次色 | #94a3b8 (slate-400) | 次要文字、标签 |

## SunnyUI 组件使用规范
- 所有 UI 组件使用 SunnyUI 命名空间下的控件
- 布局代码写在 xxx.Designer.cs 文件中
- 业务逻辑写在 xxx.cs 文件中
- 使用 UIPanel 作为容器，UICard 作为卡片容器
- 使用 UIButton、UILabel、UIDataGridView 等基础控件
